## 딕셔너리(dictionary)
>대응관계를 나타낼 수 있는 자료형

name에 '이름' birthday에 '생일'이 대응되듯이 key와 value를 한쌍으로 갖는 자료형이 딕셔너리다.

```python
>>> fruit_dict = {
    ...: '사과' : 'apple',
    ...: '복숭아': 'peach',
    ...: '자몽' : 'grapefruits'
    ...: }
```

fruit_dict의 정보

key | value
---| ---
사과 | apple
복숭아 | peach
자몽 | grapefruits
